アピールポイント
　現在制作中の本作は、プレイヤーキャラクターのアニメーションや挙動を、既存のゲームに近づけることを目指して開発しています。基礎的な移動処理に加え、SmoothDampAngle を用いた滑らかな回転処理などを実装し、より自然な動きを追求しています。また、UnityのCinemachineを活用し、カメラ演出や追従の精度を高めることで、プレイ中の没入感を強化しています。
